# code_stuff

My programming practice, mostly [USACO](https://usaco.org) Bronze solutions written in Python while I learn to code.

## USACO solutions

| File | Problem |
|------|---------|
| `traffic_code.py` | Measuring Traffic (2019 February Bronze) |
| `cowsignal_code.py` | Cow Signal |
| `Milk.py` | Mixing Milk |
| `modified_shell_game.py` | Shell Game |
| `speeding_code.py` | Speeding Ticket |
| `blist.py` | Bucket List |
| `Lostcow_code.py` | Lost Cow |

Each solution reads its input from a matching `.in` file and writes to a `.out` file, following the USACO file format.
